# ELEGSO.site (Astro)

Статический сайт юридической компании ELEGSO.
Собран на Astro в режиме static (output: "static").

Этот README используется как:
- документация проекта
- инструкция для разработчика
- ПРОМПТ ДЛЯ ИИ перед любыми изменениями

---

## Структура проекта

src/                 — исходный код сайта
src/pages/           — страницы (index, cases, slug)
src/components/      — компоненты блоков
src/layouts/         — Layout (общий каркас)
src/styles/          — стили
  global.css         — точка входа стилей
  legacy.css         — текущие рабочие стили

public/              — статика (копируется в dist как есть)
public/images/team/  — фотографии команды

dist/                — результат сборки (НЕ редактировать вручную)
.trash-bak/          — корзина проекта (архив, мусор)
.tmp/                — временные патчи и скрипты

---

## Ключевые правила проекта (ОБЯЗАТЕЛЬНО)

### Пути к изображениям
ВСЕ ссылки на изображения делаются ТОЛЬКО так:
/images/...

Почему:
- не зависят от вложенности страниц (/cases/...)
- не ломаются при смене домена
- соответствуют static-сборке Astro

НЕ использовать относительные пути вида ../images

---

## Процесс работы

Локальный запуск:
npm run dev -- --host 127.0.0.1 --port 4321

Сборка:
npm run build

Проверка dist:
dist/index.html
dist/_astro/*.css
dist/images/team/*.png
